Admonitions¶

Oftentimes, we will use admonitions (e.g. note, warning, tip) in our curriculum. Create them via RST directives.

Attention

Now is not a time to be distrated!

Caution

Something bad will happen if you ignore this!

Danger

Be careful so you don’t get hurt!

Error

Opps! Something broke.

Hint

If you want to know who ate the pizza, this might help.

Important

Some information is more important than others.

Note

Something like this might be helpful! If not, check Google

Tip

Sometimes a little help can make things click.

Warning

Things could get bad if you don’t heed this!

Generic

Sometimes, you just need to stand out.
